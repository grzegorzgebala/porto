O MNIE

Junior Web Developer

Od końca 2017 hobbystycznie rozwijam swoje umiejętności w programowaniu. Swoją naukę rozpocząłem od Java 8 ale potem miałem okazję uczestniczyć w szkoleniach Front-end i Web Developer. Podczas kursów używałem m.in. technologi HTML5, CSS, JavaScript, MongoDB, React oraz wykonywałem projekty dodatkowe - w tym jeden projekt grupowy (symulacja pracy - Jira, Kanban). Dobrze odnajduję się w zespole i zależy mi na poznawaniu nowych technologi. Jestem osobą ambitną, która lubi wyzwania i dąży do tego, aby cięgle rozwijać swoje umiejętności, dlatego swoją dalszą ścieżkę kariery chcę związać z programowaniem.
